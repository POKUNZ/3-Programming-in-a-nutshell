HTML

To start with front-end development you'll need to learn HTML so with that being said, you'll need Visual Studio Code (VS Code). Create a new file, type in HTML, and select it from the options.

Steps to Create a Basic HTML Structure:

  1. Open Visual Studio Code: Launch the application after installation.
  2. Create a New File: Click on File > New File to open a blank document.
  3. Save the File as HTML: Save the new file by clicking File > Save As and name it index.html.
  4. Type the Basic HTML Structure: In your new file, enter ! to get the following code:
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
    </body>
</html>

Explanation:

  • <!DOCTYPE html>: Declares the document type as HTML5.
  • <html lang="en">: Starts the HTML document and sets the language to English.
  • <head>: Contains meta-information like character set and title.
  • <meta charset="UTF-8">: Sets the character encoding to UTF-8.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Ensures the page is responsive on mobile devices.
  • <title>Document</title>: Sets the title of the page, displayed in the browser tab.
  • <body>: Contains the visible content of the page.
  • </html>: Closes the HTML document.

Once you've typed this code, save the file. You can then open index.html in your browser to see your first HTML page in action. Note that it is still empty because we haven't added any content yet. To add more content, check this page for further instructions.


Python

Python is a versatile programming language used for various applications, from web development to data science. Similarly to front-end development, we'll use Visual Studio Code (VS Code) for Python programming.

Key Concepts to Get Started:

  • Variables: Store information that can be used and modified later in your code.
  • Data Types: Python supports various data types like integers, floats, strings, and booleans.
  • Imports: Allows you to include and use external libraries or modules in your program.
  • Basic Syntax: Python uses indentation to define blocks of code, making it easy to read.
  • Control Structures: Use `if`, `else`, and `elif` statements to control the flow of your program.
  • Loops: Repeat actions using `for` and `while` loops.

Examples:

Variables and Data Types

# Variables
greeting = "Hello, World!"  # String
age = 25                    # Integer
pi = 3.14                   # Float
is_python_fun = True        # Boolean

print(greeting)
print("Age:", age)
print("Value of pi:", pi)
print("Is Python fun?", is_python_fun)

Imports

# Importing the math module to use mathematical functions
import math

# Using an imported function
print(math.sqrt(16))  # Outputs: 4.0

Control Structures

# Using if-else statements
number = 10

if number > 5:
    print("The number is greater than 5")
else:
    print("The number is 5 or less")

Loops

# Using a for loop
for i in range(5):
    print("Iteration:", i)

# Using a while loop
count = 0
while count < 5:
    print("Count is:", count)
    count += 1

These examples cover the basic syntax and usage of Python. For more advanced topics, like working with functions and data structures, check out this page for further instructions.


Java

Java is a versatile programming language used in a wide range of applications, including the popular game Minecraft. There is also a programming language known as GameMaker Language (GML) that shares similarities with Java. In this tutorial, we'll use Visual Studio Code (VS Code) to get started with Java programming.

Getting Started with Java

To begin programming in Java, follow these steps to set up your development environment:

  • Install the Java Development Kit (JDK): Download and install the JDK from the official Oracle website.
  • Install Visual Studio Code (VS Code): Download and install VS Code from its official website.
  • Set Up Java Extensions in VS Code: Enhance your Java development experience by installing the “Java Extension Pack” from the VS Code marketplace.

Basic Java Concepts

Here are some fundamental concepts to get you started with Java programming:

  • Variables: Store data that can be used and manipulated in your code.
  • Data Types: Java has several data types, including integers, floating-point numbers, characters, and booleans.
  • Control Flow: Use if, else, and switch statements to control the flow of your program.
  • Loops: Repeat actions using for and while loops.
  • Methods: Define reusable blocks of code. Methods can accept parameters and return values.
  • Classes and Objects: Java is an object-oriented language. Learn to create classes and objects to model real-world entities.

Variables and Data Types

Variables are used to store data. Here's an example:

public class Main {
    public static void main(String[] args) {
        String greeting = "Hello, Java!";  // String
        int age = 30;                      // Integer
        double pi = 3.14159;              // Floating-point number
        boolean isJavaFun = true;         // Boolean
        
        System.out.println(greeting);
        System.out.println("Age: " + age);
        System.out.println("Value of pi: " + pi);
        System.out.println("Is Java fun? " + isJavaFun);
    }
}

Control Flow

Control the flow of your program with conditional statements:

public class Main {
    public static void main(String[] args) {
        int number = 10;
        
        if (number > 5) {
            System.out.println("The number is greater than 5");
        } else {
            System.out.println("The number is 5 or less");
        }
    }
}

Loops

Use loops to repeat actions:

public class Main {
    public static void main(String[] args) {
        // For loop
        for (int i = 0; i < 5; i++) {
            System.out.println("Iteration: " + i);
        }
        
        // While loop
        int count = 0;
        while (count < 5) {
            System.out.println("Count is: " + count);
            count++;
        }
    }
}

These examples cover basic Java syntax. For more advanced topics, such as object-oriented programming and exception handling, refer to this page for further instructions.
