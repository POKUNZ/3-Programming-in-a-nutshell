Functions

Functions are reusable blocks of code that perform a specific task. They help in organizing code and avoiding repetition.

Defining and Calling Functions


# Define a function
def greet(name):
    return f"Hello, {name}!"

# Call the function
print(greet("Developer"))

Explanation: The greet function takes a parameter name and returns a greeting string. Functions are defined using the def keyword.

Lists and Dictionaries

Lists and dictionaries are essential data structures in Python for storing and managing collections of data.

Working with Lists


# List of items
fruits = ["apple", "banana", "cherry"]

# Accessing list items
print(fruits[0])  # Outputs: apple

# Adding an item to the list
fruits.append("date")
print(fruits)  # Outputs: ['apple', 'banana', 'cherry', 'date']

Working with Dictionaries


# Dictionary with key-value pairs
person = {
    "name": "Developer",
    "age": 25,
    "location": "Somewhere"
}

# Accessing dictionary items
print(person["name"])  # Outputs: Developer

# Adding a new key-value pair
person["hobby"] = "coding"
print(person)  # Outputs: {'name': 'Developer', 'age': 25, 'location': 'Somewhere', 'hobby': 'coding'}

Classes and Objects

Object-oriented programming (OOP) in Python allows you to define custom objects with attributes and methods.

Defining a Class


# Define a class
class GameDeveloper:
    def __init__(self, name, language):
        self.name = name
        self.language = language

    def introduce(self):
        return f"My name is {self.name} and I develop games using {self.language}."

# Create an object of the class
dev = GameDeveloper("Developer", "Python")
print(dev.introduce())  # Outputs: My name is Developer and I develop games using Python.

Explanation: The GameDeveloper class has an initializer method __init__ that sets up the object, and a method introduce that returns a formatted string.

Error Handling

Error handling is crucial for building robust applications. Python provides try-except blocks to catch and manage exceptions.

Try-Except Example


# Error handling with try-except
try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")
finally:
    print("This block runs no matter what.")

# Outputs:
# Cannot divide by zero!
# This block runs no matter what.

Explanation: The try block contains code that might throw an exception. The except block catches specific exceptions, and the finally block executes regardless of an exception occurring.

File Handling

Python allows you to work with files to read from and write to them, which is essential for data processing.

Reading and Writing Files


# Writing to a file
with open("example.txt", "w") as file:
    file.write("Hello, this is a test file.")

# Reading from a file
with open("example.txt", "r") as file:
    content = file.read()
    print(content)  # Outputs: Hello, this is a test file.

Explanation: The with statement is used to open a file. It ensures that the file is properly closed after its block of code has been executed.
