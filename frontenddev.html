Adding Content to Your HTML Page

Now that you've created the basic structure of an HTML page, it's time to add some content. Below are some common HTML elements you can use:

Headings

Headings are used to define the titles and subtitles of your content. There are six levels of headings, from <h1> (the most important) to <h6> (the least important). Here's an example:

<h1>This is a Main Heading</h1>
<h2>This is a Subheading</h2>
<h3>This is a Smaller Subheading</h3>

Paragraphs

Paragraphs are used to define blocks of text. You can create a paragraph using the <p> tag. Here's an example:

<p>This is a paragraph of text that provides information to the reader.</p>

Text Formatting

Here are some examples of text formatting:

Bold Text

Bold text: This is bold text

<b>This is bold text</b>

Italic Text

Italic text: This is italic text

<i>This is italic text</i>

Underlined Text

Underlined text: This is underlined text

<u>This is underlined text</u>

Highlighted Text

Highlighted text: This is highlighted text

<mark>This is highlighted text</mark>

Strikethrough Text

Strikethrough text: This is strikethrough text

<s>This is strikethrough text</s>

Tables

Tables are used to organize and display data in a grid format. Use the <table> tag to create a table, with <tr> for table rows, <th> for table headers, and <td> for table data. Here's an example:

<table border="1">
            <tr>
                <th>Header 1</th>
                <th>Header 2</th>
            </tr>
            <tr>
                <td>Data 1</td>
                <td>Data 2</td>
            </tr>
            <tr>
                <td>Data 3</td>
                <td>Data 4</td>
            </tr>
        </table>

This example demonstrates a basic table with two columns and three rows, including headers. You can customize tables further by using attributes such as <th/td colspan=" "> and <th/td rowspan=" "> to merge cells and create more complex layouts. Experiment with these attributes to suit your needs.

Images

To add an image to your page, use the <img> tag. The src attribute specifies the path to the image file, and the alt attribute provides alternative text in case the image cannot be displayed. Here's an example:

<img src="path/to/your-image.jpg" alt="Description of the image">

Links

To create a link to another webpage, use the <a> tag. The href attribute specifies the destination URL. Here's an example:

<a href="https://example.com">Visit Example.com</a>

Lists

You can create ordered (numbered) or unordered (bulleted) lists using <ol> and <ul> tags, respectively. Here's an example of an unordered list:

<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>

And here's an ordered list:

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Introduction to CSS

CSS (Cascading Style Sheets) is used to style HTML documents. It controls the layout, colors, fonts, and more. You can add CSS styles either internally (within your HTML file) or externally through a separate CSS file. However, using an external CSS file is generally recommended for better organization and maintainability. Ensure that both your HTML and CSS files are in the same directory. To link an external CSS file to your HTML document, put the code in between the <head> and </head> tags like this:

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <link href="yourfile.css" rel="stylesheet">
    <title>Your Title</title>
</head>

Selectors

Select elements on your page to apply styles. For example:

p {
    color: blue; /* Changes the text color of all <p> elements to blue */
}

Properties

Properties define the style you want to apply. Each property has a value. For example:

body {
    background-color: #f0f0f0; /* Light gray background */
    font-family: Arial, sans-serif; /* Font for the entire page */
}

Box Model

The CSS box model describes how elements are structured. It includes margins, borders, padding, and the content area:

div {
    margin: 10px; /* Space outside the border */
    border: 1px solid black; /* Border around the element */
    padding: 10px; /* Space inside the border */
    background-color: white; /* Background color of the element */
}

Responsive Design

CSS can be used to create responsive designs that adjust to different screen sizes. Use media queries for this:

@media (max-width: 600px) {
    body {
        font-size: 14px; /* Smaller font size for small screens */
    }
}
